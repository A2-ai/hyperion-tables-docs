## Description

Controls the number of decimal places for OFV and dOFV values. Use NA to keep significant-figure formatting.

## Usage

```r
set_spec_ofv_decimals(spec, n)
```

</tr>
<tr>

<td>Non-negative integer or NA</td>
</tr>
</tbody>
</table>

## Value

Modified spec

## Examples

```r
spec <- SummarySpec() |>
  set_spec_ofv_decimals(2)
```
