## Description

Attaches estimation method, OFV, condition number, and number of observations to parameter data for display as the first footnote in the parameter table.

## Usage

```r
add_summary_info(
  params,
  model_sum,
  show_method = TRUE,
  show_ofv = TRUE,
  show_cond_num = TRUE,
  show_number_obs = TRUE
)
```
