## Description

Controls which transforms are applied for display by parameter kind.

## Usage

```r
set_spec_transforms(spec, theta = NULL, omega = NULL, sigma = NULL)
```

<tr>

<th>Description</th>

</tr>
<tr>

</tr>
<tr>
<td><code>omega</code></td>
<td>Columns to transform for omega parameters</td>
</tr>
<tr>
<td><code>sigma</code></td>
<td>Columns to transform for sigma parameters</td>
</tr>

</table>

## Value

Modified spec

## Examples

```r
spec <- TableSpec() |>
  set_spec_transforms(theta = "all", omega = c("estimate", "cv"))
```
