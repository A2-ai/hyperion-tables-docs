## Description

Controls whether models without completed runs are excluded.

## Usage

```r
set_spec_remove_unrun(spec, remove)
```

</tr>

<tr>
<td><code>spec</code></td>
<td>A SummarySpec object</td>
</tr>
<tr>
<td><code>remove</code></td>
<td>Logical value</td>
</tr>
</tbody>
</table>
</div>

## Value

Modified spec

## Examples

```r
spec <- SummarySpec() |>
  set_spec_remove_unrun(FALSE)
```
