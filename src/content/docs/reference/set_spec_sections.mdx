## Description

Controls how parameters are grouped into sections. Pass formula expressions like `kind == "THETA" ~ "Structural Parameters"`.

## Usage

```r
set_spec_sections(spec, ..., overwrite = FALSE)
```

## Arguments

<div class="arg-table">
<table>
<thead>
<tr>
<th>Argument</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>spec</code></td>
<td>A TableSpec object</td>
</tr>
<tr>
<td><code>...</code></td>
<td>Section rule formulas</td>
</tr>
<tr>
<td><code>overwrite</code></td>
<td>If FALSE (default), append to existing rules. If TRUE, replace all existing rules.</td>
</tr>
</tbody>
</table>
</div>

## Value

Modified spec

## Examples

```r
spec <- TableSpec() |>
  set_spec_sections(
    kind == "THETA" ~ "Structural",
    kind == "OMEGA" ~ "IIV"
  )
```
