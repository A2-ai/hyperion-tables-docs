## Description

hyperion.tables provides tools for creating publication-ready parameter tables, model comparison tables, and run summary tables from NONMEM models managed by hyperion/pharos.

## Spec Classes

S7 classes for configuring table output: 
- [`TableSpec`](../tablespec/) - Configuration for parameter tables
- [`SummarySpec`](../summaryspec/) - Configuration for run summary tables
- [`CIOptions`](../cioptions/) - Confidence interval display options
- [`ParameterNameOptions`](../parameternameoptions/) - Parameter name display options

## Table Creation

High-level functions for creating tables: 
- [`make_parameter_table()`](../make_parameter_table/) - Build a formatted parameter table
- [`make_summary_table()`](../make_summary_table/) - Build a run summary table
- [`make_comparison_table()`](../make_comparison_table/) - Build a model comparison table

## Spec Application

Functions for applying specs to data: 
- [`apply_table_spec()`](../apply_table_spec/) - Enrich parameter data with transforms, CIs, sections
- [`apply_summary_spec()`](../apply_summary_spec/) - Filter models and prepare summary data
- [`apply_formatting()`](../apply_formatting/) - Apply CI merging, numeric formatting, missing text

## Spec Extraction

Functions for retrieving specs attached to data frames: 
- [`get_table_spec()`](../get_table_spec/) - Extract TableSpec from parameter data
- [`get_summary_spec()`](../get_summary_spec/) - Extract SummarySpec from summary data

## Spec Getters

Functions for reading spec properties: 
- [`get_spec_columns()`](../get_spec_columns/) - Get columns from a spec
- [`get_spec_title()`](../get_spec_title/) - Get title from a spec
- [`get_spec_sigfig()`](../get_spec_sigfig/) - Get significant figures setting
- [`get_spec_ci()`](../get_spec_ci/) - Get CI options from a TableSpec
- [`get_spec_parameter_names()`](../get_spec_parameter_names/) - Get parameter name options
- [`get_spec_sections()`](../get_spec_sections/) - Get section rules
- [`get_spec_filter()`](../get_spec_filter/) - Get row filter rules
- [`get_spec_transforms()`](../get_spec_transforms/) - Get display transforms
- [`get_spec_variability()`](../get_spec_variability/) - Get variability rules
- [`get_spec_time_format()`](../get_spec_time_format/) - Get time format from a SummarySpec

## Spec Modifiers - Columns

Functions for modifying spec column settings: 
- [`add_spec_columns()`](../add_spec_columns/) - Append columns to output
- [`drop_spec_columns()`](../drop_spec_columns/) - Exclude columns from output
- [`set_spec_columns()`](../set_spec_columns/) - Replace column list entirely

## Spec Modifiers - Common

Functions that work on both TableSpec and SummarySpec: 
- [`set_spec_title()`](../set_spec_title/) - Set table title
- [`set_spec_sigfig()`](../set_spec_sigfig/) - Set significant figures
- [`set_spec_ofv_decimals()`](../set_spec_ofv_decimals/) - Set OFV decimal places
- [`set_spec_hide_empty()`](../set_spec_hide_empty/) - Control empty column hiding
- [`set_spec_pvalue()`](../set_spec_pvalue/) - Set p-value formatting
- [`set_spec_footnotes()`](../set_spec_footnotes/) - Set footnote order

- [`set_spec_ci()`](../set_spec_ci/) - Set confidence interval options

- [`set_spec_sections()`](../set_spec_sections/) - Set section grouping rules

- [`set_spec_tag_filter()`](../set_spec_tag_filter/) - Filter by model tags
- [`set_spec_remove_unrun()`](../set_spec_remove_unrun/) - Control unrun model filtering
- [`set_spec_summary_filter()`](../set_spec_summary_filter/) - Set summary filter rules

## Rule Builders
