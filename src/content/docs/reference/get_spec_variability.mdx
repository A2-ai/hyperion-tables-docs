## Description

Returns the list of variability display rules.

## Usage

```r
get_spec_variability(spec)
```

## Arguments

<div class="arg-table">
<table>

<tr>

<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>spec</code></td>
<td>A TableSpec object</td>
</tr>
</tbody>
</table>
</div>

## Value

List of quosures

## Examples

```r
spec <- TableSpec()
get_spec_variability(spec)
```
