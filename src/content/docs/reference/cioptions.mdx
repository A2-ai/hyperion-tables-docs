## Description

Controls CI merge behavior and missing-value display.

## Usage

```r
CIOptions(
  level = 0.95,
  merge = TRUE,
  pattern = "[%s, %s]",
  missing_text = "-"
)
```

## Properties

The following properties are available on a `CIOptions` object: 
- `level` - Confidence interval level (0-1).
- `merge` - Whether to merge CI bounds into a single column.
- `pattern` - sprintf pattern for merged CI display (two %s).
- `missing_text` - Text to show for missing CI values.
